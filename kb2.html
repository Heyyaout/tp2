Ce que nous avons appris

:hover est une pseudo-classe CSS qui permet d'ajouter des effets au survol d'un élément. Par exemple, un lien ou une image peut changer de couleur, de taille ou de forme lorsqu'on passe la souris dessus.

Opacité d'une image : L'opacité permet de rendre une image plus ou moins transparente. Cela peut être utilisé pour créer des effets visuels intéressants, comme des images de fond légèrement visibles sous d'autres éléments.

Bordures arrondies : La propriété CSS border-radius permet de donner des coins arrondis aux éléments, comme les images ou les blocs de contenu.

Dégradé de fond : background-image: linear-gradient() permet de créer un dégradé de couleur, souvent utilisé comme arrière-plan pour les éléments ou le corps de la page.
